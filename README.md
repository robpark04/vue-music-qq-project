# vue-music-qq

> A Vue.js project for music
### Use
* vuejs-2.1.7
* vue-cli
* vue-router
* vuex
* muse-ui


## Support
* Chrome
* Firefox
* Mobile browser

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
